# APC Knowledge Platform - Complete Database Documentation

## Overview

This is a comprehensive database package for the **APC Nigeria Electoral & Political Knowledge Platform** (apcng.org). It contains 600 quiz questions, 1000+ political facts & quotes, complete administrative divisions, electoral data, and INEC information.

### 1. **apc_political_quiz.json** (42KB)
**600 Multiple Choice Questions** across 6 categories

#### Categories:
- **APC Overview & Structure** (80Q) - Formation, electoral history, leadership
- **Nigerian Political History** (120Q) - Independence through Fourth Republic
- **Nigerian Contemporary Politics** (150Q) - Current governance, structures
- **Global Politics & International Context** (150Q) - UN, AU, ECOWAS, international frameworks
- **APC Policies & Governance** (100Q) - Policy initiatives, governance record
- **Electoral Systems & Processes** (100Q) - Voting mechanics, candidate requirements

#### Structure:
```json
{
  "id": 1,
  "question": "What does APC stand for?",
  "options": ["All Progressives Congress", "African Political Coalition", ...],
  "correct": 0,
  "difficulty": "easy|medium|hard",
  "explanation": "APC is the All Progressives Congress...",
  "category": "APC Overview & Structure"

```

#### Usage:

```
State ‚Üí LGA (Local Government Area) ‚Üí Ward
```

- **36 States** (one JSON each)

#### File Format:

## üîó Integration Guide for apcng.org

### Frontend Setup (React)

```javascript
// 1. Import all datasets
import quizData from './apc_political_quiz.json';
import factsData from './apc_political_facts_quotes.json';
import adminData from './nigeria_administrative_divisions_complete.json';
import electoralData from './nigerian_electoral_system.json';
import masterIndex from './apc_master_index.json';

// 2. Create data service
export const dataService = {
  getRandomQuiz: (category) => {
    const categoryData = quizData.quiz.categories[category];
    return categoryData.questions[Math.floor(Math.random() * categoryData.questions.length)];
  },
  
  getStateData: (stateCode) => {
    return adminData.nigeriaAdministrativeDivisions.states
      .find(s => s.id === stateCode);
  },
  
  getWardsByLGA: (lgaCode, stateCode) => {
    const state = adminData.nigeriaAdministrativeDivisions.states
      .find(s => s.id === stateCode);
    const lga = state.lgas.find(l => l.id === lgaCode);
    return lga.wards;
  },
  
  getSenatorialZones: (state) => {
    return electoralData.electoralSystem.senatorialDistricts
      .filter(s => s.state === state);
  }
};
```

### Backend Setup (Node.js/Express)

```javascript
// 1. Load datasets into server memory or database
const quizData = require('./apc_political_quiz.json');
const adminData = require('./nigeria_administrative_divisions_complete.json');

// 2. Create API routes
app.get('/api/quiz/:category', (req, res) => {
  const category = req.params.category;
  const questions = quizData.quiz.categories[category].questions;
  res.json(questions.slice(0, 10)); // Return first 10
});

app.get('/api/states', (req, res) => {
  res.json(adminData.nigeriaAdministrativeDivisions.statesSummary);
});

app.get('/api/state/:code/lgas', (req, res) => {
  const state = adminData.nigeriaAdministrativeDivisions.states
    .find(s => s.id === req.params.code);
  res.json(state.lgas);
});
```

### Mobile App (React Native/Flutter)

```javascript
// SQLite setup for offline ward data
const db = SQLite.openDatabase('apc_app.db');

// Download and cache admin divisions locally
const initializeDatabase = async () => {
  const adminData = await fetch('/api/admin-divisions');
  db.tx(t => {
    adminData.states.forEach(state => {
      db.run('INSERT INTO states (id, name, region) VALUES (?, ?, ?)',
        [state.id, state.name, state.region]);
    });
  });
};
```
